Dept. of Computer Science, Courant Institute of Mathematical Science, New York University

Introduction

We introduce DropConnect, a generalization of Hinton's Dropout for regularizing large fully-connected layers within neural networks. When training with Dropout, a randomly selected subset of activations are set to zero within each layer. DropConnect instead sets a randomly selected subset of weights within the network to zero. Each unit thus receives input from a random subset of units in the previous layer. We derive a bound on the generalization performance of both Dropout and DropConnect.

Motivation

Training Network with Dropout:
Each element of a layer's output is kept with probability pp, otherwise being set to 00 with probability 1−p1−p. If we further assume neural activation function with a(0)=0a(0)=0, such as tanhtanh and relurelu (⋆⋆ is element-wise multiplication ):
r=m⋆a(Wv)=a(m⋆Wv)r=m⋆a(Wv)=a(m⋆Wv)
Training Network with DropConnect:
Generalization of Dropout in which each connection, rather than each output unit, can be dropped with probability 1−p1−p:
r=a((M⋆W)v)r=a((M⋆W)v)
where MM is weight mask, WW is fully-connected layer weights and vv is fully-connected layer inputs.

Mixture Model Interpretation

DropConnect Network is a mixture model of 2|M|2|M| neural network classifiers f(x;θ,M)f(x;θ,M):
o=EM[f(x;θ,M)]=∑Mp(M)f(x;θ,M)o=EM[f(x;θ,M)]=∑Mp(M)f(x;θ,M)
It is not hard to show stochastic gradient descent with random mask MM for each data improves the lower bound of mixture model

Inference

Dropout Network Inference (mean-inference): EM[a(M⋆W)v]≈a(EM[(M⋆W)v])=a(pWv)EM[a(M⋆W)v]≈a(EM[(M⋆W)v])=a(pWv)
DropConnect Network Inference (sampling): EM[a(M⋆W)v]≈Eu[a(u)]EM[a(M⋆W)v]≈Eu[a(u)] where u∼N(pWv,p(1−p)(W⋆W)(v⋆v))u∼N(pWv,p(1−p)(W⋆W)(v⋆v)), i.e. each neuron activation are approximated by a Gaussian distribution via moment matching.

Why DropConnect Regularize Network

Rademacher Complexity of Model: max|Ws|≤Bsmax|Ws|≤Bs, max|W|≤Bmax|W|≤B, kk is the number of classes, R^ℓ(G)R^ℓ(G) is the Rademacher complexity of the feature extractor, nn and dd are the dimensionality of the input and output of the DropConnect layer respectively:
R^ℓ(F)≤p(2k√dBsnd√Bh)R^ℓ(G)R^ℓ(F)≤p(2kdBsndBh)R^ℓ(G)

Special Cases of pp:
  1. p=0p=0: the model complexity is zero, since the input has no influence on the output.
  2. p=1p=1: it returns to the complexity of a standard model.
  3. p=1/2p=1/2: all sub-models have equal preference.
